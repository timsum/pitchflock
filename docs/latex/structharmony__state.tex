\doxysubsection{Detailed Description}
Represents the state of a harmonic structure. 

This struct contains information about the encoded state, chromatic notes, and KPDVE (Key, Pattern, Degree, Voicing, Extension) values. It also includes lists for possible KPDVE encodings and their associated distances.
